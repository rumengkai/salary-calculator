# 中国个税计算器微信小程序

一款功能全面的中国个人所得税计算工具，基于微信小程序原生框架开发，支持多种计税场景。

## 功能特性

- **月度工资个税** — 累计预扣预缴法，展示 12 个月逐月明细
- **年度汇算** — 综合所得（工资+劳务+稿酬+特许权）合并计税
- **年终奖计税对比** — 单独计税 vs 并入综合所得，智能推荐最优方案
- **五险一金配置** — 输入缴费基数和比例，自动计算月缴金额
- **专项附加扣除配置** — 7 项扣除（子女教育、继续教育、住房贷款/租金、赡养老人、婴幼儿照护）
- **个税知识** — 税率表、计算公式、政策 FAQ

## 设计特点

- 毛玻璃 UI 风格（渐变背景 + 半透明卡片 + 圆角）
- 全局配置：五险一金和专项扣除只需配置一次，各计算页面自动读取
- 输入缓存：所有页面输入数据通过 localStorage 持久化
- 浮动计算按钮，适配安全区域

## 技术栈

- 框架：微信小程序原生（WXML + WXSS + JS）
- 测试：Jest（88 个测试用例，分支覆盖率 93%+）
- 无第三方 UI 库依赖

## 税率规则

基于中国现行个人所得税法（2019 年起实施）：

- 综合所得：7 级超额累进税率（3% ~ 45%）
- 起征点：5000 元/月（60000 元/年）
- 年终奖单独计税：按月度税率表（÷12 找税率）
- 专项附加扣除标准：2023 年 8 月提标后最新标准
